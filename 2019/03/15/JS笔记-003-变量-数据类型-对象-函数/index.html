JS笔记-003-变量-数据类型-对象-函数

发表于 2019-03-15

声明变量：不管是局部还是全局变量都是用var声明就可以

var x =2;
var y =3;
var z = x+y;
var pi=3.14;
var name="Bill Gates";//使用引号包括就是字符串了
var answer='Yes I am!';//可以使用单引号包括

字符串要使用引号包括，里面可以嵌套，可以整型拼接字符串写入,写入也可以直接使用整型

var answer="Nice to meet you!";
var answer="He is called 'Bill'";
var answer='He is called "Bill"';

创建一个数组的两种方式

var cars=new Array();
cars[0]="Audi";
cars[1]="BMW";
cars[2]="Volvo";
var cars=new Array("Audi","BMW","Volvo");

创建一个对象

var person={firstname:"Bill", lastname:"Gates", id:5566};

从对象里面取值有，get方法

name=person.lastname;
name=person["lastname"];

使用var创建对象，可以规定数据类型

var carname=new String;
var x=      new Number;
var y=      new Boolean;
var cars=   new Array;
var person= new Object;

js函数，就是封装方法。function 必须是小写

function functionname()
{
这里是要执行的代码
}

返回值使用reture返回，使用注意，因为没有规定返回的数据类型，所以方法会比较容易重复

function myFunction()
{
var x=5;
return x;
}

返回值可以省略return（以后证明）

function myFunction(a,b)
{
if (a>b)
  {
  return;
  }
x=a+b
}

如果一个变量没有提前使用var去声明，而是直接赋值的话，就直接作为全局变量

carname="Volvo";
